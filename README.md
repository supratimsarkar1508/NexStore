# SpentWise - Personal Expense Tracker

SpentWise is a modern, responsive React application designed to help users track their personal finances. Record income and expenses, categorize spending, and visualize financial habits with interactive charts.

## Features

-   **Dashboard Overview**: Instant view of your balance, total income, and total expenses.
-   **Transaction Management**: Add and delete transactions with ease.
-   **Categorization**: Automatic categorization to see where your money goes.
-   **Data Visualization**: Pie charts for category breakdown and Bar charts for cash flow analysis.
-   **Persistence**: All data is saved to your browser's LocalStorage, so it stays even after refreshing.
-   **Responsive Design**: Fully functional on mobile, tablet, and desktop screens.
-   **Search & Filter**: Quickly find specific transactions by description or category.

## Technologies Used

-   **React 18**: Frontend library.
-   **TypeScript**: For type-safe development.
-   **Tailwind CSS**: Modern utility-first CSS framework.
-   **Recharts**: Composable charting library.
-   **Lucide React**: Beautiful & consistent icons.
-   **Context API**: State management for global transaction data.

## Prerequisites

-   Node.js (v16.0.0 or higher)
-   npm or yarn
